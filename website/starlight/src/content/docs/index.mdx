## Requirements

- **vcpkg**: recommend [2025.12.12](https://github.com/microsoft/vcpkg/releases/tag/2025.12.12) or later
- **vcpkg-tool**: recommend [2025-12-16](https://github.com/microsoft/vcpkg-tool/releases/tag/2025-12-16) or later

## Features

This registry provides:

- Custom overlay ports for specialized libraries
- Custom triplets for Android NDK and iOS Simulator
- Comprehensive documentation for port creation and maintenance
- GitHub Copilot prompts for automated workflows

## Setup

Follow the official [Getting Started Guide](https://learn.microsoft.com/en-us/vcpkg/get_started/get-started) to setup your vcpkg environment, then clone this registry:

```bash
export VCPKG_ROOT="/usr/local/share/vcpkg"
pushd $VCPKG_ROOT
    git clone https://github.com/luncliff/vcpkg-registry
popd
```

For more detailed setup instructions, check out the [Create Port Guide](/vcpkg-registry/guides/create-port/).
