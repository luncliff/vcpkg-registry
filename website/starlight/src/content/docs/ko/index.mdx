## 기능

이 레지스트리는 다음을 제공합니다:

- 특수 라이브러리를 위한 사용자 정의 오버레이 포트
- Android NDK 및 iOS 시뮬레이터용 사용자 정의 트리플릿
- 포트 생성 및 유지 관리를 위한 포괄적인 문서
- 자동화된 워크플로를 위한 GitHub Copilot 프롬프트

## 설정

공식 [시작 가이드](https://learn.microsoft.com/ko-kr/vcpkg/get_started/get-started)를 따라 vcpkg 환경을 설정한 다음 이 레지스트리를 복제하세요:

```bash
export VCPKG_ROOT="/usr/local/share/vcpkg"
pushd $VCPKG_ROOT
    git clone https://github.com/luncliff/vcpkg-registry
popd
```

자세한 설정 지침은 [포트 생성 가이드](/vcpkg-registry/ko/guides/create-port/)를 확인하세요.
